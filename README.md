# 🎂 Age Calculator

A modern and responsive Age Calculator web application developed as part of the **CodeAlpha Web Development Internship**.

## 🚀 Project Overview

This Age Calculator allows users to enter their Date of Birth and instantly calculate their exact age in:

* Years
* Months
* Days

The project is built using HTML, CSS, and JavaScript and demonstrates the use of DOM Manipulation, JavaScript Date Object, and Input Validation.

---

## ✨ Features

✅ Calculate age in Years, Months, and Days

✅ User-friendly and responsive interface

✅ Date of Birth selection using Day, Month, and Year dropdowns

✅ Input validation for incorrect dates

✅ Future date restriction

✅ Birthday countdown feature

✅ Total Months, Days, Hours, and Minutes calculation

✅ Reset functionality
